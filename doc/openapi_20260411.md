## Как проект строит URL

`application/lib/ts/send.js` и `application/lib/ts/stream.js` принимают endpoint без
версии и добавляют `ver` сами:

- v3 default: `endpoint: ['marketdata', 'quotes', 'MSFT']` →
  `/v3/marketdata/quotes/MSFT`.
- v2 explicitly: `ver: 'v2', endpoint: ['data', 'symbols', 'suggest', text]` →
  `/v2/data/symbols/suggest/{text}`.
- stream endpoints читаются как JSON lines через `fetch(...).body.getReader()`.

## Symbol rules from snapshot

OpenAPI examples use TradeStation option symbols with a space before expiration:
`MSFT 220916C305`.

Project internal OPT symbol format is compact:
`MSFT220916C305000` for a 305 strike with millesimal strike storage.
`lib.utils.makeTSSymbol(symbol, 'OPT')` must preserve full `YYMMDD`:

```text
CRWV281201C80000 -> CRWV 281201C80
FSLR260618C270000 -> FSLR 260618C270
GLNG270115C55000 -> GLNG 270115C55
FSLR261218C270000 -> FSLR 261218C270
```

Do not trim leading zeroes from month or day. `CRWV 28121C80` is invalid because
the expiration part has 5 digits instead of 6.

For raw URLs the space in option symbols must be encoded as `%20`. In this project,
query strings go through `URLSearchParams`; path values are converted by `new URL(...)`.

## Stream contract notes

TradeStation stream responses are JSON line packets. `application/lib/ts/stream.js`
must classify packets before publishing data:

- `Heartbeat` means the stream is alive; the snapshot says heartbeat is sent after
  5 seconds on an idle stream.
- `StreamStatus: 'GoAway'` or `Error: 'GoAway'` means server shutdown / stream
  handoff; reconnect is expected.
- `StreamStatus: 'EndSnapshot'` appears in brokerage streams after the initial
  snapshot; it is not a failure.
- `StreamErrorResponse` shape is `{ Error, Message }`.
- Quote error packets can also include `{ Symbol, Error }`, for example
  `{ Symbol: 'INVALID', Error: 'FAILED, INVALID SYMBOL' }`.
- `INVALID SYMBOL` is permanent for the given subscription. Do not reconnect to the
  same upstream stream without a new client request or corrected symbol.
- `serializeError` and stream `onError` handlers must prefer `Error`, then
  `Message`, then `message`, not `String(object)`.

## Response shape notes

High-signal schemas to check before changing parsers:

- `Bars` → object with `Bars: Bar[]`.
- `Bar` → prices and volumes are mostly strings; flags include `IsRealtime`,
  `IsEndOfHistory`, `BarStatus`.
- `QuoteSnapshot` → object with optional `Quotes: Quote[]` and `Errors: QuoteError[]`.
- `QuoteError` → `{ Symbol, Error }`.
- `QuoteStream` → quote fields plus optional `Error`.
- `Spread` → option chain / option quote stream payload; contains greeks, prices,
  `Side`, `Strikes`, `Legs`.
- `SpreadLeg` → `{ Symbol, Ratio, StrikePrice, Expiration, OptionType, AssetType }`.
- `Expirations` → `Expirations: Expiration1[]`.
- `Expiration1` → `{ Date, Type }`.
- `Strikes` → `{ SpreadType, Strikes: string[][] }`.
- `SpreadTypes` → `SpreadTypes: SpreadType[]`.
- `RiskRewardAnalysisInput` → `{ SpreadPrice, Legs }`.
- `RiskRewardAnalysisResult` → max gain/loss flags and breakeven points.
- `SymbolDetailsResponse` → `Symbols[]` and `Errors[]`; max 50 symbols.
- `StreamStatus` → `{ StreamStatus }`; known values include `EndSnapshot`, `GoAway`.

## Implementation implications

- Use `DomainError` for predictable unsupported project modes, for example unsupported
  option `spreadType` in `application/api/options/chain.js`.
- `riskFreeRate` exists in the TradeStation upstream `GetOptionChain` contract, but
  `ts_connect` currently disables it in `application/api/options/chain.js` and does
  not include it in the downstream request or stream key.
- Use `Error` for transport failures, parse failures and broken integration invariants.
- Keep stream state in `application/domain/ts/streams.js`, not in `application/api/*`.
- New managed stream endpoints should follow `application/api/stream/addBarchart.js`
  and `application/api/options/chain.js`.
- `GET /v3/marketdata/stream/options/quotes` is not the same endpoint as
  `GET /v3/marketdata/stream/quotes/{symbols}`. The former takes `legs[n].Symbol`
  query params and returns spread greeks; the latter streams quote changes for
  comma-separated symbols.
- A preflight options board request can be a defensive guard, but it is not a
  replacement for correct internal → TradeStation symbol conversion.
- `GET /v2/stream/tickbars/{symbol}/{interval}/{barsBack}` is locally treated as a bounded JSON-lines stream: `symbol`, `interval` (1-64999), and `barsBack` (1-10) are validated before the stream is read into a packet array with `api.start`, `ts.request.done`, and `api.done` diagnostics.
